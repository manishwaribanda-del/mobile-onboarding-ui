# 📱 Mobile Onboarding Flow UI

A clean and modern mobile onboarding flow designed to create a smooth and engaging first-time user experience. This concept focuses on simplicity, usability, and visually pleasing aesthetics using [...]

---

## ✨ Screens Included

### 👋 Welcome Screen
Introduces users to the application with a friendly illustration and welcoming message.

### 📋 Planning Screen
Highlights task organization, goal setting, and productivity features.

### 📈 Progress Screen
Encourages users to track achievements, stay consistent, and remain motivated.

---

## 🎨 Design Highlights

- Minimal and modern UI design
- Soft pastel purple color palette
- User-friendly onboarding experience
- Clean typography and balanced spacing
- Mobile-first responsive layout
- Smooth visual flow between screens
- Modern CTA buttons and onboarding indicators
